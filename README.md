# Homebrew Tap For mcp-broker

Install:

```bash
brew tap NavinAgrawal/tap
brew install mcp-broker
```

This tap packages the same Python CLI published to PyPI:

- `mcp-broker`
- `mcp-broker-client`
- `mcp-broker-daemon`

The formula installs the CLI only. It does not edit Codex, Claude, Cursor, or
other client configuration during install.
